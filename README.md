# Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript. This project showcases my profile, education, skills, work experience, projects, and contact information in a clean and interactive layout.

## Features

* About Me section with profile image
* Education details
* Work Experience section
* Technical Skills section
* Projects showcase
* Contact Information
* Interactive JavaScript functionality:

  * Show/Hide About Me content
  * Welcome Message button
  * Skills Counter
  * Theme Changer
* Responsive design for different screen sizes

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## Sections Included

* About Me
* Education
* Work Experience
* Technical Skills
* Projects
* Contact Information

## Purpose

This project was created to strengthen my understanding of front-end web development concepts, including page structure, styling, responsiveness, and JavaScript DOM manipulation.

## Learning Outcomes

Through this project, I practiced:

* Semantic HTML
* CSS Styling and Layout Design
* Responsive Web Design
* JavaScript Functions
* DOM Manipulation
* Event Handling
